# 0x03-思路整理
https://github.com/r0eXpeR/GetInfo

### 旁站
目标未使用CDN时，可通过nmap扫描目标全端口，查看开放哪些其他端口，通过其他端口切入
### C段
fofa收集目标子域，通过查看子域对应IP，可得知目标的大概C段
### ASN
当目标组织较大时，可考虑收集目标所在ASN的全部资产
### 历史IP
通过ip138.com查看目标域名解析过的历史IP
